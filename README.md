# Madhav-Ecommerce-Sales-Dashboard-
Built an interactive E-Commerce Sales Dashboard in Power BI to analyze sales, profit, customers, and regional performance. Used Power Query for data cleaning and DAX for KPIs. Created interactive dashboards to generate business insights and support data-driven decision-making.

## Objective
Analyze sales performance using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX

## KPIs
- Total Sales
- Total Profit
- Quantity Sold
